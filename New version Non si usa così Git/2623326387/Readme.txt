This mod makes the wires and bearings easier to see. It adds a dark contour on wires, bearings and connections to make them more visible. 


HOW TO INSTALL:
-Goto:
	"...\Steam\SteamApps\workshop\content\387990\2623326387"
-Select the folder depending on the game version (0.5 Version or 0.6 TestVersion)
-Copy the Data folder into:
	"...\Steam\SteamApps\common\Scrap Mechanic" and replace the exsiting files.
-ONLY For the Test version goto:
	"...\Steam\SteamApps\common\Scrap Mechanic\cache" 
	and delete texture and mesh folders.

HOW TO UNINSTALL:
The easier way is to use the steam tool to "Verify integrity of game files"
If you want to restore the original files manually
-Go to:
	"...\Steam\SteamApps\workshop\content\387990\2623326387\Backup"
-Select the folder depending on game version and copy the Data folder into:
	"...\Steam\SteamApps\common\Scrap Mechanic"
	(replacing the exsisting files).


HOW TO CHANGE BEARINGS TO SMALL
-After the normal installation, go to:
	"...\Steam\SteamApps\common\Scrap Mechanic\Data\Mesh"
-Rename the file "bearing_connect.mesh" to something else like "bearing_connect_standard.mesh" (if you don't see the ".mesh" at the end of the name, just ignore that part).
-Rename the file "bearing_connect_small.mesh" to "bearing_connect.mesh".

HOW TO CHANGE ALTERNATIVE MOTORPOINT AND WIRES
-After the normal installation, go to:
	"...\Steam\SteamApps\common\Scrap Mechanic\Data\Textures\ConnectTool"
-Rename the files like with the bearings choosing the texture you want.
-Remember to delete cache folder if you already launched the game.

HOW TO CHANGE UNCONNECTED BEARINGS
This feature is experimental, the unconnected bearing uses a "generic" texture that could be used by other things.
We could not find any broken texture with this mod applied, if you do please let us know!
-To install, copy transparent.tga from the folder:
	"...\Steam\SteamApps\workshop\content\387990\2623326387\NotConnectedBearing"
-Paste it in:
	"...\Steam\SteamApps\common\Scrap Mechanic\Data\Texture"
