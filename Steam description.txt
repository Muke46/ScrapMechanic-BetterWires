This mod makes the wires and bearings easier to see. It adds a dark contour on wires, bearings and connection points to make them more visible. 


[h1] HOW TO INSTALL: [/h1]
[list]
[*]Go to:
"...\Steam\SteamApps\workshop\content\387990\2623326387\Game-Version"
[*] Copy the Data folder into:
"...\Steam\SteamApps\common\Scrap Mechanic"
	and replace the exsiting files.
[*]Go to:
 "...\Steam\SteamApps\common\Scrap Mechanic\Cache" 
and delete Texture and Mesh folders.
[/list]


[h1] HOW TO CHANGE ALTERNATIVE TEXTURES AND MASHES FOR BEARINGS, WIRES AND DOTS [/h1]

The unconnected bearing is kind of "experimental" as his texture looks like generic,so could be used by other things.
We could not find any broken texture with this mod applied, if you do please let us know!

After the normal installation:
[list]
[*]Go to:
"...\Steam\SteamApps\workshop\content\387990\2623326387\Game-Version\Alternative"
[*]Choose what you want replace, then copy choosed folder's content to:
"...\Steam\SteamApps\common\Scrap Mechanic"
	and replace the existing files.
[*]Go to:
 "...\Steam\SteamApps\common\Scrap Mechanic\Cache" 
and delete Texture and Mesh folders.
[/list]


[h1] HOW TO UNINSTALL: [/h1]
The easier way is to use the steam tool to "Verify integrity of game files".

If you want to restore the original files manually:
[list]
[*]Go to:
"...\Steam\SteamApps\workshop\content\387990\2623326387\Game-Version\Backup"
[*]Select the folder depending on game version and copy the Data folder into:
"...\Steam\SteamApps\common\Scrap Mechanic"
	and replace the exsisting files.
[*]Go to:
 "...\Steam\SteamApps\common\Scrap Mechanic\Cache" 
and delete Texture and Mesh folders.
[/list]
